# 개요
대회 기능에 사용되는 DB 테이블 목록을 나열함.


# 테이블
### contest
|컬럼명|타입|제약조건|컬럼설명|
|---|---|---|---|
|id|BIGINT|PK, GENERATED BY DEFAULT AS IDENTITY|고유-대회번호, 자동증가|
|created_by|BIGINT|FK → domain_user.id, NOT NULL, ON DELETE CASCADE|대회 생성자 고유-유저번호|
|group_id|BIGINT|FK → domain_group.id, ON DELETE SET NULL|그룹 공개 대회일 때 연결 그룹, nullable|
|title|VARCHAR(255)|NOT NULL|대회 제목|
|description|TEXT||대회 소개|
|contest_type|VARCHAR(20)|NOT NULL|OFFICIAL / GROUP / CLASS / INDIVIDUAL|
|participation_type|VARCHAR(20)|NOT NULL|INDIVIDUAL / TEAM|
|visibility|VARCHAR(20)|NOT NULL|PUBLIC / GROUP|
|start_time|TIMESTAMP|NOT NULL|대회 시작 시각|
|end_time|TIMESTAMP|NOT NULL|대회 종료 시각|
|rules|TEXT||참가 규칙 및 주의사항|
|prize_description|VARCHAR(500)||시상 내용|
|is_hot|BOOLEAN|NOT NULL, DEFAULT FALSE|인기 대회 여부|
|poster_url|VARCHAR(500)||대회 포스터 이미지 URL|
|created_at|TIMESTAMP|NOT NULL, DEFAULT CURRENT_TIMESTAMP|생성 시점|
|updated_at|TIMESTAMP|NOT NULL, DEFAULT CURRENT_TIMESTAMP|수정 시점|

### contest_tag

| 컬럼명                     | 타입          | 제약조건                                         | 컬럼설명            |
| ----------------------- | ----------- | -------------------------------------------- | --------------- |
| id                      | BIGINT      | PK, GENERATED BY DEFAULT AS IDENTITY         |                 |
| contest_id              | BIGINT      | FK → contest.id, NOT NULL, ON DELETE CASCADE |                 |
| tag                     | VARCHAR(50) | NOT NULL                                     |                 |
| UNIQUE(contest_id, tag) |             |                                              | 동일 대회에 중복 태그 방지 |

### contest_problem

| 컬럼명                            | 타입     | 제약조건                                                 | 컬럼설명              |
| ------------------------------ | ------ | ---------------------------------------------------- | ----------------- |
| id                             | BIGINT | PK, GENERATED BY DEFAULT AS IDENTITY                 |                   |
| contest_id                     | BIGINT | FK → contest.id, NOT NULL, ON DELETE CASCADE         |                   |

| UNIQUE(contest_id, problem_id) |        |                                                      | 한 대회에 동일 문제 중복 방지 |

### contest_participant

| 컬럼명                         | 타입          | 제약조건                                             | 컬럼설명                |
| --------------------------- | ----------- | ------------------------------------------------ | ------------------- |
| id                          | BIGINT      | PK, GENERATED BY DEFAULT AS IDENTITY             |                     |
| contest_id                  | BIGINT      | FK → contest.id, NOT NULL, ON DELETE CASCADE     |                     |
| user_id                     | BIGINT      | FK → domain_user.id, NOT NULL, ON DELETE CASCADE |                     |
| role                        | VARCHAR(20) | NOT NULL, DEFAULT 'PARTICIPANT’                  | ADMIN / PARTICIPANT |
| joined_at                   | TIMESTAMP   | NOT NULL, DEFAULT CURRENT_TIMESTAMP              | 참가 시점               |
| is_banned                   | BOOLEAN     | NOT NULL, DEFAULT FALSE                          | 실격 여부               |
| score                       | INT         | NOT NULL, DEFAULT 0                              | 스코어보드용 누적 점수        |
| last_solved_at              | TIMESTAMP   |                                                  | 마지막 정답 시각, nullable |
| UNIQUE(contest_id, user_id) |             |                                                  | 동일 대회 중복 참가 방지      |

### contest_submission

| 컬럼명                | 타입        | 제약조건                                                  | 컬럼설명         |
| ------------------ | --------- | ----------------------------------------------------- | ------------ |
| id                 | BIGINT    | PK, GENERATED BY DEFAULT AS IDENTITY                  |              |
| contest_id         | BIGINT    | FK → contest.id, NOT NULL, ON DELETE CASCADE          |              |
| contest_problem_id | BIGINT    | FK → contest_problem.id, NOT NULL, ON DELETE CASCADE  |              |
| user_id            | BIGINT    | FK → domain_user.id, NOT NULL, ON DELETE CASCADE      |              |
| submission_id      | BIGINT    | FK → solve_submission.id, NOT NULL, ON DELETE CASCADE | 기존 채점 시스템 연결 |
| is_correct         | BOOLEAN   | nullable                                              | 정답 여부        |
| submitted_at       | TIMESTAMP | NOT NULL                                              | 제출 시각        |

> `(contest_problem_id, user_id)` 정답 제출은 1회만 점수 인정 → 서비스 레이어에서 처리

### contest_qna
